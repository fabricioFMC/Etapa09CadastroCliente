Objetivo do software:
É digitalizar e padronizar o processo de coleta e armazenamento de informações dos clientes, 
servindo como a fonte primária de dados para outros sistemas e setores da empresa, 
como Vendas, Marketing e Suporte.

Funcionalidades do sistema
RF001: O sistema deve permitir o cadastro completo de um novo cliente

RF002: O sistema deve permitir a edição/atualização de todos os dados de um cliente existente, respeitando as regras de negócio.

RF003: O sistema deve permitir a busca de clientes por Nome, CPF/CNPJ e Telefone.

RF004: O sistema deve alertar o usuário sobre a duplicidade de CPF ou CNPJ durante o cadastro.

RF005: O sistema deve permitir o registro de notas e histórico de contato por cliente.

RF006: O sistema deve permitir a exportação da lista de clientes em formato CSV.

RF007: O sistema deve permitir a inativação de um registro de cliente.
